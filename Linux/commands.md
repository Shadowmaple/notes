# Linux命令集

基于 Ubuntu 18.04

## 基础

+   cat
+   ls
+   cd
+   more
+   less
+   head：从头部按行读取
+   tail：从尾部按行读取
+   tree
+   cal：日历
+   history
+   mv
+   cp
+   rm
+   find
+   locate
+   whereis
+   which
+   grep
+   ln：创建链接

## 压缩

+   tar
+   unzip
+   zip
+   gzip

## 进程服务

+   systemctl：系统服务
+   jobs：查看当前shell的任务
+   fg
+   bg
+   top：动态进程查看
+   ps：静态进程查看
+   nohup
+   netstat
